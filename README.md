# Odoo + PrestaShop Connector

Conector bidireccional entre Odoo 16 y PrestaShop 9 mediante Docker Compose.

## Requisitos

- Docker Desktop instalado y abierto
- Puerto 8069 y 8080 libres

## Metodo de instalacion

descomprimir el .zip
buscar el archivo start.bat y hacerle doble clic 

## Encender y apagar servicios (siguientes veces)
en la aplicacion de docker desktop se te generara un container
llamado odoo-prestashop-podoks en action podras pararlo y encenderlo

**NUNCA usar `docker compose down` ya que borra los datos.**
